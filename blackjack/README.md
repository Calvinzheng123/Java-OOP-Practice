# Blackjack

A console-based Java project built to practice object-oriented programming by simulating a simplified blackjack game.

## Features
- Create and shuffle a deck of cards
- Deal cards to player and dealer
- Hit or stand gameplay
- Dealer follows blackjack rules and hits until at least 17
- Determine the winner based on final hand values

## OOP Concepts Used
- Encapsulation
- Inheritance
- Composition
- Abstraction
- Enums

## Class Structure
- Card
- Deck
- Hand
- Player
- Dealer
- Suit (enum)
- Rank (enum)
- BlackjackGame
- Main
