# Task Manager

A console-based Java project built to practice object-oriented programming by managing tasks.

## Features

- Add tasks with title, description, and priority
- View all tasks
- Mark tasks as completed
- Delete tasks by ID

## OOP Concepts Used

- Encapsulation
- Abstraction
- Composition
- Separation of concerns
- Enums

## Class Structure

- Task
- Priority (enum)
- Status (enum)
- TaskManager
- Main
