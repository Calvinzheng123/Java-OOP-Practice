# Parking Lot System

A console-based Java project built to practice object-oriented programming by simulating a parking lot.

## Features
- Park cars, trucks, and motorcycles
- Assign vehicles to valid parking spots
- Remove vehicles by license plate
- Display available and occupied spots

## OOP Concepts Used
- Encapsulation
- Inheritance
- Composition
- Abstraction
- Enums

## Class Structure
- Vehicle (abstract)
- Car / Truck / Motorcycle
- ParkingSpot
- ParkingLot
- Ticket
- ParkingService

## Future Improvements / challenges to add
- Multiple parking levels
- Hourly billing
